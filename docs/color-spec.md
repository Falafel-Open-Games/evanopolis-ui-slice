# Evanopolis UI Slice - Pawn Color Spec

## Color Pairs

- charcoal -> silver
- tomato-jam -> salmon
- autumn-leaf -> honey-bronze
- fern -> celadon
- bright-ocean -> maya-blue
- amethyst -> violet

## Usage Rules

- 6 player slots, including charcoal/silver as a valid player color.
- Players choose any of the six colors, non-repeating.
- No special "current player" color.
- Text over dark colors: white.
- Text over light colors: black.

## City Group Palette (Monochrome)

These colors are for city groups only. Do not use them for player ownership.
Names come from Coolors; note that "charcoal" here is a city shade and is
different from the player "charcoal" above.

- Caracas -> graphite: #2b2b2b
- Assunción -> gunmetal: #404040
- Ciudad del Este -> charcoal: #595959
- Minsk -> dim-grey: #737373
- Irkutsk -> grey-olive: #8f8f8f
- Rockdale -> silver: #adadad
